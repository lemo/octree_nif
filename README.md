octree_nif
==========

A minimalistic octree nif for erlang using PointCloud octree as underlying structure.

License
-------
Released under BSD-License to stay compatible to PointCloud library.

Dependencies
-----

* rebar (obviously)
* eigen (needed by pcl-octree)
* pcl-octree 
* nifpp 
 

Usage
-----

```bash
rebar co
```
